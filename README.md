# Reception Management System
A Laravel & Livewire application for managing office front-desk operations including call logging, visitor tracking, and employee reviews.

## Features
- **Call Logging:** Track inbound/outbound calls with role-based access.
- **Visitor Management:** Digital logbook for checking guests in and out.
- **Employee Directory:** Manage staff details and internal reviews.

## Tech Stack
- **Framework:** Laravel 11
- **Frontend:** Livewire 3, TailwindCSS, Blade
- **Auth:** Laravel Jetstream
